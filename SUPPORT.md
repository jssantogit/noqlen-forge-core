# Support

For bugs or feature requests, use GitHub Issues in `jssantogit/noqlen-forge-core`.

Noqlen Forge Core is early. PyPI may not be available yet, so use the installation instructions in the README rather than assuming package registry availability.

## What To Include

For usage questions or bug reports, include safe details only:

- Operating system.
- Python version.
- `noqlen-forge` command used.
- Sanitized error output.
- Sanitized config snippets if needed.
- Whether the command was dry-run or apply mode.

## What Not To Include

- Secrets, tokens, passwords, API keys, salts, or cookies.
- Real local paths or real music-library paths.
- Complete lyric text or raw fingerprints.
- Private metadata dumps, provider payloads, database exports, or server backups.
- Logs that reveal private library structure or credentials.

For security-sensitive reports, do not paste secrets or private data into public issues. Open a minimal public issue asking for a private reporting path, or contact the maintainer through a trusted private channel if one is available.
