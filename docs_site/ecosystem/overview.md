# Ecosystem Overview

Noqlen is the ecosystem name for local music-library workflows. The goal is controlled metadata, import, server, and playback tooling that stays reviewable and privacy-conscious.

## Current Public Component

Noqlen Forge Core is the current public component and the main documented product area. It focuses on metadata, library management, reports, review, playlists, ratings, and Navidrome-oriented workflows.

## Planned Context

- **Noqlen Flux Core** is planned for downloads and import workflows.
- **Noqlen Anchor Core** is planned for Navidrome and local server management.
- **Noqlen Core** is expected to emerge gradually as shared foundations become stable.
- **Noqlen Aria** is a future player and app experience.

These planned components are ecosystem direction, not current dependencies or implemented products documented by this site.
