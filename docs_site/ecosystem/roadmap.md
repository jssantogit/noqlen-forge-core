# Ecosystem Roadmap

No dates are promised. The sequence below describes product focus, not release commitments.

## Current Focus: Forge

Stabilize Noqlen Forge Core as the CLI-first metadata and library-management foundation. Improve safety, review, reports, configuration clarity, and real-library guidance.

## Next: Flux

Noqlen Flux Core is planned to cover downloads and import workflows once Forge is stable enough to be the reliable library side of that process.

## Then: Anchor

Noqlen Anchor Core is planned for Navidrome and local server management after the library and import foundations are clearer.

## Later: Aria

Noqlen Aria is a future player and app experience. It should come after the underlying library, import, and server workflows are mature.

## Shared Foundations

Noqlen Core should emerge gradually as stable shared foundations become obvious across the ecosystem. It should not be extracted prematurely.
